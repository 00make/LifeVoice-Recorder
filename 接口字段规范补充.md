# 接口字段规范补充（LifeOS）

## 1. 通用字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| event_id | uuid | 事件 ID |
| event_type | text | 事件类型 |
| source | text | 数据来源 |
| timestamp | timestamptz | 事件时间 |

## 2. 常用 payload 字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| segment_id | text | 语音片段 ID |
| path | text | 存储路径 |
| hash | text | SHA-256 |
| duration_ms | int | 时长 |
| model_version | text | 模型版本 |
| pipeline_version | text | 流水线版本 |
